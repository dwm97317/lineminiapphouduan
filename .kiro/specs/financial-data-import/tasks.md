# 实现计划: 财务原始数据导入 (修订版)

## 概述

本实现计划将财务原始数据导入功能分解为一系列增量式的编码任务。该功能用于从带有颜色标记的Excel文件中导入历史订单支付状态，包括Excel解析、颜色检测、订单匹配、状态更新和预览界面等核心模块。

**重要修订**：基于实际数据库结构，`yoshop_inpack` 表没有直接的 `express_num` 字段，需要通过 `pack_ids` 关联 `yoshop_package` 表查询国际单号。

实现策略采用自底向上的方式，先构建核心服务层组件，然后实现控制器层，最后完成前端交互界面。每个任务都包含具体的代码实现目标和需求追溯。
