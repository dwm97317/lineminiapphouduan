## 数据库架构

### 1. 面单打印记录表 (`yoshop_waybill_record`)

存储每次面单打印或下单的记录。

**主要字段：**
- `id` - 主键
- `inpack_id` - 集运订单ID
- `order_sn` - 订单号
- `express_type` - 快递公司类型（zhongtong/shunfeng）
- `express_name` - 快递公司名称
- `waybill_no` - 运单号
- `operation_type` - 操作类型（1:打印 2:只下单）
- `operator_id` / `operator_name` - 操作人信息
- `print_time` - 打印时间
- `api_response` - API响应数据（JSON）
- `created_time` - 创建时间

### 2. 配置数据存储 (`yoshop_setting`)

使用系统设置表存储三类配置：

**a) API 配置** (`key = 'waybill'`)
```json
{
  "zhongtong": {
    "api_url": "中通API地址",
    "api_key": "API Key",
    "api_secret": "API Secret",
    "company_code": "ZTO"
  },
  "shunfeng": {
    "api_url": "顺丰API地址",
    "api_key": "Partner ID",
    "api_secret": "Checkword",
    "custid": "月结卡号",
    "pay_method": "付款方式",
    "company_code": "SF"
  }
}
```

**b) 中通面单配置** (`key = 'waybill_config_zhongtong'`)
```json
{
  "fields": {
    "sender_name": true,
    "sender_phone": true,
    "sender_address": true,
    "receiver_name": true,
    "receiver_phone": true,
    "receiver_address": true,
    "item_name": true,
    "weight": true,
    "volume": false,
    "remark": false,
    "quantity": true
  },
  "company_fields": {
    "site_code": "网点代码",
    "site_name": "网点名称"
  },
  "print_params": {
    "paper_size": "76x130",
    "orientation": "portrait",
    "scale": 100
  }
}
```

**c) 顺丰面单配置** (`key = 'waybill_config_shunfeng'`)
```json
{
  "fields": { ... },
  "company_fields": {
    "monthly_card": "月结卡号",
    "payment_method": "1"
  },
  "print_params": { ... }

```

主要功能：

- 实时保存和读取配置
- 恢复默认配置

## 使用方法

### 1. 初始化数据库

运行初始化脚本：
```bash
php init_waybill_database.php
```

这将自动：
- 创建 `yoshop_waybill_record` 表
- 初始化 API 配置
- 初始化中通和顺丰的面单配置

### 2. 访问配置页面

在浏览器中访问：
```
http://your-domain/index.php?s=/store/setting.waybill_config/index
```

### 3. 配置 API 信息

**中通快递配置：**
1. 填写 API 地址（如：`https://api.zhongtong.com`）
2. 填写 API Key
3. 填写 API Secret
4. 点击"保存 API 配置"

**顺丰快递配置：**
1. 填写 API 地址
   - 生产环境：`https://bspgw.sf-express.com/std/service`
   - 沙箱环境：`https://sfapi-sbox.sf-express.com/std/service`
2. 填写合作伙伴编码 (partnerID)
3. 填写校验码/密钥 (checkword)
4. 填写月结卡号 (custid，可选)
5. 选择默认付款方式
6. 点击"保存 API 配置"

### 4. 配置面单显示

1. 在"面单配置"选项卡中选择快递公司
2. 勾选需要显示的字段
3. 填写快递公司特定字段（如中通的网点代码）
4. 设置打印参数（纸张大小、方向、缩放比例）
5. 点击"保存配置"

### 5. 恢复默认配置

如需恢复默认设置，点击"恢复默认"按钮。

## API 接口说明

### 获取 API 配置
```
GET /store/setting.waybill_config/getApiConfig
```

### 保存 API 配置
```
POST /store/setting.waybill_config/saveApiConfig
参数: config (JSON字符串或对象)
```

### 获取面单配置
```
GET /store/setting.waybill_config/getConfig
参数: express_type (zhongtong/shunfeng)
```

### 保存面单配置
```
POST /store/setting.waybill_config/saveConfig
参数: 
  - express_type (快递类型)
  - config (配置对象/JSON字符串)
```

### 获取字段列表
```
GET /store/setting.waybill_config/getFieldList
参数: express_type (zhongtong/shunfeng)
```

### 恢复默认配置
```
POST /store/setting.waybill_config/resetConfig
参数: express_type (快递类型)
```

## 注意事项

1. **数据库前缀**：所有表都使用 `yoshop_` 前缀，与系统配置保持一致

2. **wxapp_id**：所有配置数据默认使用 `wxapp_id = 10001`，根据实际情况调整

3. **缓存管理**：配置保存后会自动清除缓存，确保立即生效

4. **JSON 格式**：所有配置值都以 JSON 格式存储在 `values` 字段中

5. **字段验证**：保存配置时会进行基本验证，确保必需字段存在

## 扩展功能

如需添加新的快递公司支持：

1. 在 `WaybillConfigService::getDefaultConfig()` 中添加公司默认配置
2. 在 `WaybillConfigService::getFieldDefinitions()` 中添加字段定义
3. 在视图中添加对应的配置选项
4. 更新 `getExpressTypes()` 方法添加新的快递公司选项

## 常见问题

### Q: 配置保存后没有生效？
A: 检查缓存是否已清除。可以手动清空 `runtime/cache` 目录。

### Q: 出现数据库连接错误？
A: 检查 `source/application/database.php` 中的数据库配置是否正确。

### Q: 如何查看面单打印记录？
A: 查询 `yoshop_waybill_record` 表，或通过订单管理页面查看。
