# 历史订单支付状态批量导入

**负责人**：后端开发E  
**预计工期**：1天  
**依赖**：数据库设计完成

## 一、功能背景

财务有一份Excel原始数据表，记录了所有集运订单的信息。在这份表格中使用三种颜色标记订单状态：
- **蓝色背景**：已支付 + 已发送账单
- **粉红色背景**：已发送账单（未支付）
- **绿色背景**：已支付（未发送账单）
- **无背景色**：未处理

**问题**：
- 旧系统中订单的`is_pay`字段不准确
- 财务需要手动核对哪些订单已支付、已出账
- 新系统上线后，需要快速将历史订单状态同步到系统中

**解决方案**：
导入财务的原始Excel表格，系统自动识别不同颜色的行，根据颜色执行不同的更新操作：
1. 蓝色：标记已支付 + 创建历史账单
2. 粉红色：创建历史账单（保持未支付）
3. 绿色：标记已支付（不创建账单）
4. 这些订单将按正确的状态进入新系统，避免重复处理

**功能特点**：
- 这是一次性功能，仅用于历史数据导入
- 采用简单规则 + 灵活处理的方案
- 支持部分成功，按客户分批处理
- 提供完整的预览和人工确认机制

## 二、Excel文件格式

### 2.1 文件结构

**多工作表结构**：
- Excel文件包含多个Sheet（工作表）
- 每个Sheet对应一个或多个客户的订单记录
- Sheet命名通常为：客户ID、客户名称或日期

**示例文件结构**：
```
Excel文件：财务原始数据.xlsx
├── Sheet1: 客户A
├── Sheet2: 客户B
├── Sheet3: 2月份订单
└── Sheet4: 其他
```

**每个Sheet的列结构**：
- **A列：用户ID + 国家**（如：23048泰国、5809越南）
- **B列：重量**（KG）
- **C列：国际单号**（express_num，可能为空）
- **D列：日期**（可选，如：2月13日）

**颜色标记含义**：
- **蓝色背景**：已支付 + 已发送账单
- **粉红色背景**：已发送账单（未支付）
- **绿色背景**：已支付（未发送账单）
- **白色/无背景**：未处理

**示例Sheet内容**：
```
| A列        | B列  | C列              | D列     | 说明           |
|-----------|------|------------------|---------|----------------|
| 23048泰国  | 1.8  | 73596298537719   |         | (蓝色背景)      |
| 25831泰国  | 5.2  | 73596298743207   |         | (蓝色背景)      |
| 23273泰国  | 3    | 73596298668051   |         | (粉红色背景)    |
| 30424泰国  | 1.2  | 73596298668049   |         | (绿色背景)      |
| 5809越南   | 2.6  | SF512221791891   | 2月13日 | (绿色背景)      |
| 5809泰国   | 29.6 |                  |         | (绿色背景，无国际单号) |
| 17136泰国  | 4.2  | 73596298537721   |         | (无背景)        |
```

**特殊情况**：
1. 用户ID前面可能有图片
2. 国际单号可能为空（需要通过用户ID+重量+日期匹配）
3. 用户ID包含国家后缀（需要提取数字部分）

### 2.2 订单匹配方式

**方式1：通过国际单号精确匹配（优先）**
- 使用C列的国际单号（express_num）
- 对应系统中集运订单的`express_num`字段
- 格式：文本或数字

**方式2：通过用户ID + 重量 + 日期模糊匹配**
- 当国际单号为空时使用
- 提取A列的用户ID（去除国家后缀）
- 使用B列的重量（允许±0.5KG误差）
- 使用D列的日期范围（如果有）
- 优先匹配未出账、未支付的订单

**用户ID提取规则**：
- 从"23048泰国"中提取"23048"
- 从"5809越南"中提取"5809"
- 忽略前面的图片内容

### 2.3 多Sheet处理

**处理逻辑**：
1. 读取Excel文件的所有Sheet
2. 遍历每个Sheet
3. 识别每个Sheet中着色的行
4. 提取所有着色行的运单号
5. 批量更新订单状态

**Sheet选择**：
- 默认：处理所有Sheet
- 可选：让用户选择要处理的Sheet

### 3.3 颜色识别算法

**简单RGB阈值判断**：

```php
/**
 * 检测单元格背景色
 */
private function detectCellColor($sheet, $row)
{
    // 检查A列或B列的背景色
    foreach (['A', 'B'] as $col) {
        $cell = $sheet->getCell($col . $row);
        $style = $cell->getStyle();
        $fill = $style->getFill();
        $fillType = $fill->getFillType();
        $fillColor = $fill->getStartColor()->getRGB();
        
        if ($fillType == \PhpOffice\PhpSpreadsheet\Style\Fill::FILL_NONE 
            || !$fillColor || $fillColor == 'FFFFFF') {
            continue;
        }
        
        // 转换为RGB值
        $r = hexdec(substr($fillColor, 0, 2));
        $g = hexdec(substr($fillColor, 2, 2));
        $b = hexdec(substr($fillColor, 4, 2));
        
        // 蓝色：B值明显高于R和G
        if ($b > 200 && $b > $r + 30 && $b > $g + 30) {
            return [
                'color' => 'blue',
                'confidence' => 'high',
                'rgb' => "$r,$g,$b"
            ];
        }
        
        // 粉红色：R值高，G和B中等
        if ($r > 200 && $g > 150 && $b > 150) {
            return [
                'color' => 'pink',
                'confidence' => 'high',
                'rgb' => "$r,$g,$b"
            ];
        }
        
        // 绿色：G值明显高于R和B
        if ($g > 200 && $g > $r + 30 && $g > $b + 30) {
            return [
                'color' => 'green',
                'confidence' => 'high',
                'rgb' => "$r,$g,$b"
            ];
        }
        
        // 有颜色但无法识别
        return [
            'color' => 'unknown',
            'confidence' => 'low',
            'rgb' => "$r,$g,$b"
        ];
    }
    
    return [
        'color' => 'white',
        'confidence' => 'high',
        'rgb' => '255,255,255'
    ];
}
```

**处理策略**：
- 高置信度：直接归类到对应颜色
- 无法识别：在预览界面显示RGB值，让财务手动选择颜色

### 3.4 订单匹配策略

**匹配优先级**：

```php
/**
 * 匹配订单
 */
private function matchPackage($memberId, $weight, $internationalNo, $date)
{
    // 方式1：通过国际单号精确匹配（优先）
    if (!empty($internationalNo)) {
        $package = Package::where('express_num', $internationalNo)
            ->where('wxapp_id', $this->wxapp_id)
            ->where('is_delete', 0)
            ->find();
        
        if ($package) {
            return [
                'package' => $package,
                'match_type' => 'exact',
                'confidence' => 'high'
            ];
        }
    }
    
    // 方式2：通过用户ID + 重量 + 日期模糊匹配
    $dateRange = $this->getDateRange($date);
    
    // 查找所有候选订单
    $candidates = Package::where('member_id', $memberId)
        ->where('wxapp_id', $this->wxapp_id)
        ->where('is_delete', 0)
        ->where(function($query) use ($weight) {
            // 重量误差±0.5KG
            $query->where('weight', '>=', $weight - 0.5)
                  ->where('weight', '<=', $weight + 0.5);
        })
        ->where('entering_warehouse_time', '>=', $dateRange['start'])
        ->where('entering_warehouse_time', '<=', $dateRange['end'])
        ->select();
    
    if (empty($candidates)) {
        return null;  // 无法匹配
    }
    
    // 计算每个候选订单的重量差异
    foreach ($candidates as &$candidate) {
        $candidate['weight_diff'] = abs($candidate['weight'] - $weight);
    }
    
    // 按重量差异排序
    usort($candidates, function($a, $b) {
        return $a['weight_diff'] <=> $b['weight_diff'];
    });
    
    // 如果最接近的有多个（重量差异相同）
    $minDiff = $candidates[0]['weight_diff'];
    $bestMatches = array_filter($candidates, function($c) use ($minDiff) {
        return $c['weight_diff'] == $minDiff;
    });
    
    if (count($bestMatches) > 1) {
        // 需要人工选择
        return [
            'package' => null,
            'candidates' => $bestMatches,
            'match_type' => 'multiple',
            'confidence' => 'low'
        ];
    }
    
    // 返回最接近的订单
    return [
        'package' => $candidates[0],
        'match_type' => 'fuzzy',
        'confidence' => 'medium'
    ];
}
```

**匹配结果**：
- `exact`：通过国际单号精确匹配（高置信度）
- `fuzzy`：通过用户ID+重量匹配（中置信度）
- `multiple`：有多个候选订单（低置信度，需人工选择）
- `null`：无法匹配

### 3.5 历史账单创建策略

**按客户分别创建历史账单**：

```php
/**
 * 为每个客户创建历史账单
 */
private function createHistoryStatements($results)
{
    $statements = [];
    
    // 按客户分组（蓝色和粉红色需要创建账单）
    $needStatementOrders = array_merge($results['blue'], $results['pink']);
    $groupedByMember = [];
    
    foreach ($needStatementOrders as $item) {
        if ($item['package']) {
            $memberId = $item['package']['member_id'];
            if (!isset($groupedByMember[$memberId])) {
                $groupedByMember[$memberId] = [];
            }
            $groupedByMember[$memberId][] = $item;
        }
    }
    
    // 为每个客户创建历史账单
    foreach ($groupedByMember as $memberId => $orders) {
        $member = Member::find($memberId);
        
        $statement = Statement::create([
            'statement_no' => 'HISTORY_' . $memberId . '_' . date('YmdHis'),
            'member_id' => $memberId,
            'member_name' => $member ? $member['name'] : '未知客户',
            'start_date' => date('Y-m-01'),
            'end_date' => date('Y-m-t'),
            'status' => Statement::STATUS_CONFIRMED,
            'pay_status' => Statement::PAY_STATUS_PAID,  // 历史账单默认已支付
            'remark' => '历史数据导入',
            'wxapp_id' => $this->wxapp_id
        ]);
        
        $statements[$memberId] = $statement['id'];
    }
    
    return $statements;
}
```

**账单特点**：
- 账单编号：`HISTORY_{客户ID}_{导入时间}`
- 账单状态：已确认
- 支付状态：已支付
- 每个客户一个历史账单

### 3.6 批量更新策略

**按客户分批处理，独立事务**：

```php
/**
 * 批量更新订单状态
 */
public function batchUpdateFromRawData($results, $statements)
{
    $successCount = 0;
    $failedCount = 0;
    $failedDetails = [];
    
    // 按客户分组所有订单
    $groupedByMember = [];
    foreach (['blue', 'pink', 'green'] as $color) {
        foreach ($results[$color] as $item) {
            if ($item['package']) {
                $memberId = $item['package']['member_id'];
                if (!isset($groupedByMember[$memberId])) {
                    $groupedByMember[$memberId] = [
                        'blue' => [],
                        'pink' => [],
                        'green' => []
                    ];
                }
                $groupedByMember[$memberId][$color][] = $item;
            }
        }
    }
    
    // 按客户分批处理
    foreach ($groupedByMember as $memberId => $orders) {
        Db::startTrans();
        try {
            $statementId = $statements[$memberId] ?? null;
            
            // 处理蓝色（已支付已出账）
            foreach ($orders['blue'] as $item) {
                Package::where('id', $item['package']['id'])
                    ->update([
                        'is_pay' => 1,
                        'pay_time' => date('Y-m-d H:i:s'),
                        'statement_id' => $statementId
                    ]);
                $successCount++;
            }
            
            // 处理粉红色（已出账未支付）
            foreach ($orders['pink'] as $item) {
                Package::where('id', $item['package']['id'])
                    ->update([
                        'statement_id' => $statementId
                    ]);
                $successCount++;
            }
            
            // 处理绿色（已支付未出账）
            foreach ($orders['green'] as $item) {
                Package::where('id', $item['package']['id'])
                    ->update([
                        'is_pay' => 1,
                        'pay_time' => date('Y-m-d H:i:s')
                    ]);
                $successCount++;
            }
            
            Db::commit();
            
        } catch (\Exception $e) {
            Db::rollback();
            $failedCount += count($orders['blue']) + count($orders['pink']) + count($orders['green']);
            $failedDetails[] = [
                'member_id' => $memberId,
                'error' => $e->getMessage()
            ];
        }
    }
    
    return [
        'success_count' => $successCount,
        'failed_count' => $failedCount,
        'failed_details' => $failedDetails
    ];
}
```

**处理特点**：
- 按客户分批，每个客户独立事务
- 某个客户失败不影响其他客户
- 记录详细的失败信息

// 辅助函数：提取用户ID
function extractMemberId($raw) {
    $text = strip_tags($raw);  // 去除图片
    $text = trim($text);
    
    // 提取数字部分
    if (preg_match('/^(\d+)/', $text, $matches)) {
        return intval($matches[1]);
    }
    
    return null;
}

// 辅助函数：检测背景色
function detectCellColor($sheet, $row) {
    foreach (['A', 'B'] as $col) {
        $cell = $sheet->getCell($col . $row);
        $style = $cell->getStyle();
        $fill = $style->getFill();
        $fillColor = $fill->getStartColor()->getRGB();
        
        if (!$fillColor || $fillColor == 'FFFFFF') {
            continue;
        }
        
        // 根据RGB值判断颜色
        if (isBlueColor($fillColor)) {
            return 'blue';
        } elseif (isPinkColor($fillColor)) {
            return 'pink';
        } elseif (isGreenColor($fillColor)) {
            return 'green';
        }
    }
    
    return 'white';
}

// 辅助函数：匹配订单
function matchPackage($memberId, $weight, $internationalNo, $date) {
    // 方式1：通过国际单号精确匹配
    if (!empty($internationalNo)) {
        $package = Package::where('express_num', $internationalNo)
            ->where('is_delete', 0)
            ->find();
        
        if ($package) {
            return $package;
        }
    }
    
    // 方式2：通过用户ID + 重量模糊匹配
    $dateRange = getDateRange($date);
    
    $package = Package::where('member_id', $memberId)
        ->where('is_delete', 0)
        ->where(function($query) use ($weight) {
            // 重量误差±0.5KG
            $query->where('weight', '>=', $weight - 0.5)
                  ->where('weight', '<=', $weight + 0.5);
        })
        ->where('entering_warehouse_time', '>=', $dateRange['start'])
        ->where('entering_warehouse_time', '<=', $dateRange['end'])
        ->order('entering_warehouse_time', 'desc')
        ->find();
    
    return $package;
}

### 3.7 安全机制

**两阶段提交**：
1. 阶段1：解析预览，不修改数据库
2. 阶段2：财务确认后才执行更新

**部分成功机制**：
- 按客户分批处理，每个客户独立事务
- 某个客户失败不影响其他客户
- 记录详细的失败信息和原因

**数据验证**：
1. 检查订单是否存在
2. 检查订单是否属于当前商家
3. 检查订单是否已删除
4. 验证用户ID是否匹配（警告但不阻止）

**颜色识别容错**：
- 高置信度：自动归类
- 无法识别：显示RGB值，让财务手动选择
- 记录所有识别结果供后续分析

**订单匹配容错**：
- 精确匹配：通过国际单号
- 模糊匹配：通过用户ID+重量+日期
- 多个候选：显示列表，让财务选择
- 无法匹配：单独列出，可导出处理

### 4.1 服务层（PaymentImportService.php）

**文件路径**：`source/application/common/service/PaymentImportService.php`

```php
<?php
namespace app\common\service;

use app\common\model\Package;
use PhpOffice\PhpSpreadsheet\IOFactory;

class PaymentImportService
{
    private $wxapp_id;
    
    public function __construct($wxapp_id = null)
    {
        $this->wxapp_id = $wxapp_id ?: helper::getWxappId();
    }
    
    /**
     * 解析Excel文件，提取不同颜色的订单（支持多Sheet）
     */
    public function parseExcelFile($filePath)
    {
        $spreadsheet = IOFactory::load($filePath);
        $allSheets = $spreadsheet->getAllSheets();
        
        $results = [
            'blue' => [],      // 已支付已出账
            'pink' => [],      // 已出账未支付
            'green' => [],     // 已支付未出账
            'unmatched' => []  // 无法匹配
        ];
        
        $sheetResults = [];
        
        // 遍历所有Sheet
        foreach ($allSheets as $sheet) {
            $sheetName = $sheet->getTitle();
            $highestRow = $sheet->getHighestRow();
            
            $sheetStats = [
                'blue' => 0,
                'pink' => 0,
                'green' => 0,
                'white' => 0
            ];
            
            for ($row = 2; $row <= $highestRow; $row++) {
                // 提取数据
                $memberIdRaw = trim($sheet->getCell('A' . $row)->getValue());
                $weight = floatval($sheet->getCell('B' . $row)->getValue());
                $internationalNo = trim($sheet->getCell('C' . $row)->getValue());
                $date = $sheet->getCell('D' . $row)->getValue();
                
                // 提取用户ID（去除国家后缀）
                $memberId = $this->extractMemberId($memberIdRaw);
                
                if (!$memberId || !$weight) {
                    continue;  // 跳过无效数据
                }
                
                // 检测背景色
                $color = $this->detectCellColor($sheet, $row);
                
                if ($color == 'white') {
                    $sheetStats['white']++;
                    continue;  // 跳过无背景色的行
                }
                
                $sheetStats[$color]++;
                
                // 匹配订单
                $package = $this->matchPackage($memberId, $weight, $internationalNo, $date);
                
                $item = [
                    'sheet_name' => $sheetName,
                    'row' => $row,
                    'member_id' => $memberId,
                    'weight' => $weight,
                    'international_no' => $internationalNo,
                    'date' => $date,
                    'color' => $color,
                    'package' => $package
                ];
                
                if ($package) {
                    $results[$color][] = $item;
                } else {
                    $results['unmatched'][] = $item;
                }
            }
            
            $sheetResults[] = [
                'sheet_name' => $sheetName,
                'total_rows' => $highestRow - 1,
                'blue_count' => $sheetStats['blue'],
                'pink_count' => $sheetStats['pink'],
                'green_count' => $sheetStats['green'],
                'white_count' => $sheetStats['white']
            ];
        }
        
        return [
            'total_sheets' => count($allSheets),
            'sheet_results' => $sheetResults,
            'results' => $results,
            'summary' => [
                'blue_count' => count($results['blue']),
                'pink_count' => count($results['pink']),
                'green_count' => count($results['green']),
                'unmatched_count' => count($results['unmatched'])
            ]
        ];
    }
    
    /**
     * 提取用户ID（去除国家后缀）
     */
    private function extractMemberId($raw)
    {
        // 去除图片、空格等
        $text = strip_tags($raw);
        $text = trim($text);
        
        // 提取数字部分（用户ID）
        if (preg_match('/^(\d+)/', $text, $matches)) {
            return intval($matches[1]);
        }
        
        return null;
    }
    
    /**
     * 检测单元格背景色
     */
    private function detectCellColor($sheet, $row)
    {
        // 检查A列或B列的背景色
        foreach (['A', 'B'] as $col) {
            $cell = $sheet->getCell($col . $row);
            $style = $cell->getStyle();
            $fill = $style->getFill();
            $fillType = $fill->getFillType();
            $fillColor = $fill->getStartColor()->getRGB();
            
            if ($fillType == \PhpOffice\PhpSpreadsheet\Style\Fill::FILL_NONE 
                || !$fillColor || $fillColor == 'FFFFFF') {
                continue;
            }
            
            // 根据RGB值判断颜色
            if ($this->isBlueColor($fillColor)) {
                return 'blue';
            } elseif ($this->isPinkColor($fillColor)) {
                return 'pink';
            } elseif ($this->isGreenColor($fillColor)) {
                return 'green';
            }
        }
        
        return 'white';
    }
    
    /**
     * 判断是否为蓝色
     */
    private function isBlueColor($rgb)
    {
        // 蓝色系：B值较高，R和G值较低
        $r = hexdec(substr($rgb, 0, 2));
        $g = hexdec(substr($rgb, 2, 2));
        $b = hexdec(substr($rgb, 4, 2));
        
        return $b > 200 && $b > $r && $b > $g;
    }
    
    /**
     * 判断是否为粉红色
     */
    private function isPinkColor($rgb)
    {
        // 粉红色系：R值较高，G和B值中等
        $r = hexdec(substr($rgb, 0, 2));
        $g = hexdec(substr($rgb, 2, 2));
        $b = hexdec(substr($rgb, 4, 2));
        
        return $r > 200 && $g > 150 && $b > 150;
    }
    
    /**
     * 判断是否为绿色
     */
    private function isGreenColor($rgb)
    {
        // 绿色系：G值较高
        $r = hexdec(substr($rgb, 0, 2));
        $g = hexdec(substr($rgb, 2, 2));
        $b = hexdec(substr($rgb, 4, 2));
        
        return $g > 200 && $g > $r && $g > $b;
    }
    
    /**
     * 匹配订单
     */
    private function matchPackage($memberId, $weight, $internationalNo, $date)
    {
        // 方式1：通过国际单号精确匹配
        if (!empty($internationalNo)) {
            $package = Package::where('express_num', $internationalNo)
                ->where('wxapp_id', $this->wxapp_id)
                ->where('is_delete', 0)
                ->find();
            
            if ($package) {
                return $package;
            }
        }
        
        // 方式2：通过用户ID + 重量模糊匹配
        $dateRange = $this->getDateRange($date);
        
        $package = Package::where('member_id', $memberId)
            ->where('wxapp_id', $this->wxapp_id)
            ->where('is_delete', 0)
            ->where(function($query) use ($weight) {
                // 重量误差±0.5KG
                $query->where('weight', '>=', $weight - 0.5)
                      ->where('weight', '<=', $weight + 0.5);
            })
            ->where('entering_warehouse_time', '>=', $dateRange['start'])
            ->where('entering_warehouse_time', '<=', $dateRange['end'])
            ->order('entering_warehouse_time', 'desc')
            ->find();
        
        return $package;
    }
    
    /**
     * 获取日期范围
     */
    private function getDateRange($date)
    {
        if ($date) {
            // 解析日期（如：2月13日）
            if (preg_match('/(\d+)月(\d+)日/', $date, $matches)) {
                $month = intval($matches[1]);
                $day = intval($matches[2]);
                $year = date('Y');
                
                return [
                    'start' => "$year-$month-$day 00:00:00",
                    'end' => "$year-$month-$day 23:59:59"
                ];
            }
        }
        
        // 默认：当月
        return [
            'start' => date('Y-m-01 00:00:00'),
            'end' => date('Y-m-t 23:59:59')
        ];
    }
    
    /**
     * 预览将要更新的订单（支持多Sheet）
     */
    public function previewUpdate($coloredOrders)
    {
        // 提取所有运单号
        $orderNumbers = array_column($coloredOrders, 'order_no');
        
        $packages = Package::where('express_num', 'in', $orderNumbers)
            ->where('wxapp_id', $this->wxapp_id)
            ->where('is_delete', 0)
            ->select();
        
        $result = [
            'found' => [],
            'not_found' => [],
            'already_paid' => [],
            'already_billed' => [],
            'member_mismatch' => []  // 用户ID不匹配
        ];
        
        $foundOrders = [];
        foreach ($packages as $package) {
            $foundOrders[] = $package['express_num'];
            
            // 查找对应的Excel数据
            $excelData = null;
            foreach ($coloredOrders as $item) {
                if ($item['order_no'] == $package['express_num']) {
                    $excelData = $item;
                    break;
                }
            }
            
            // 检查用户ID是否匹配
            if ($excelData && $excelData['member_id'] && $excelData['member_id'] != $package['member_id']) {
                $package['excel_member_id'] = $excelData['member_id'];
                $package['sheet_name'] = $excelData['sheet_name'];
                $result['member_mismatch'][] = $package;
                continue;
            }
            
            if ($package['is_pay'] == 1) {
                $result['already_paid'][] = $package;
            } elseif ($package['statement_id']) {
                $result['already_billed'][] = $package;
            } else {
                $package['sheet_name'] = $excelData ? $excelData['sheet_name'] : '';
                $result['found'][] = $package;
            }
        }
        
        // 找出不存在的订单号
        $result['not_found'] = array_diff($orderNumbers, $foundOrders);
        
        return $result;
    }
    
    /**
     * 批量更新订单支付状态
     */
    public function batchUpdatePaymentStatus($orderNumbers, $remark = '历史数据导入')
    {
        // 只更新未支付且未出账的订单
        $affectedRows = Package::where('express_num', 'in', $orderNumbers)
            ->where('wxapp_id', $this->wxapp_id)
            ->where('is_pay', 0)
            ->where('statement_id', null)
            ->where('is_delete', 0)
            ->update([
                'is_pay' => 1,
                'pay_time' => date('Y-m-d H:i:s'),
                'admin_remark' => $remark
            ]);
        
        // 记录导入日志
        $this->logImport($orderNumbers, $affectedRows);
        
        return $affectedRows;
    }
    
    /**
     * 记录导入日志
     */
    private function logImport($orderNumbers, $affectedRows)
    {
        // 可以记录到日志表或文件
        $logData = [
            'type' => 'payment_import',
            'order_count' => count($orderNumbers),
            'affected_rows' => $affectedRows,
            'order_numbers' => implode(',', $orderNumbers),
            'import_time' => date('Y-m-d H:i:s'),
            'wxapp_id' => $this->wxapp_id
        ];
        
        // 写入日志
        \think\Log::info('Payment Import: ' . json_encode($logData));
    }
}

    /**
     * 上传并解析Excel文件
     */
    public function upload()
    {
        $file = $this->request->file('file');
        
        if (!$file) {
            return $this->renderError('请选择文件');
        }
        
        // 验证文件类型
        $ext = strtolower($file->getOriginalExtension());
        if (!in_array($ext, ['xls', 'xlsx'])) {
            return $this->renderError('仅支持Excel格式（.xls或.xlsx）');
        }
        
        // 保存临时文件
        $savePath = './uploads/temp/';
        if (!is_dir($savePath)) {
            mkdir($savePath, 0755, true);
        }
        
        $info = $file->move($savePath);
        $filePath = $savePath . $info->getSaveName();
        
        try {
            $service = new PaymentImportService();
            $parseResult = $service->parseExcelFile($filePath);
            
            // 提取订单号
            $orderNumbers = array_column($parseResult['colored_orders'], 'order_no');
            
            // 预览将要更新的订单
            $preview = $service->previewUpdate($orderNumbers);
            
            // 保存文件路径到session，供确认导入时使用
            session('import_file_path', $filePath);
            session('import_order_numbers', $orderNumbers);
            
            return $this->renderSuccess('解析成功', [
                'parse_result' => $parseResult,
                'preview' => $preview,
                'summary' => [
                    'total_colored' => count($orderNumbers),
                    'can_update' => count($preview['found']),
                    'already_paid' => count($preview['already_paid']),
                    'already_billed' => count($preview['already_billed']),
                    'not_found' => count($preview['not_found'])
                ]
            ]);
            
        } catch (\Exception $e) {
            return $this->renderError('解析失败：' . $e->getMessage());
        }
    }
    
    /**
     * 确认导入
     */
    public function confirm()
    {
        $orderNumbers = session('import_order_numbers');
        
        if (empty($orderNumbers)) {
            return $this->renderError('请先上传文件');
        }
        
        try {
            $service = new PaymentImportService();
            $affectedRows = $service->batchUpdatePaymentStatus($orderNumbers);
            
            // 清除session
            session('import_file_path', null);
            session('import_order_numbers', null);
            
            return $this->renderSuccess('导入成功', [
                'affected_rows' => $affectedRows
            ]);
            
        } catch (\Exception $e) {
            return $this->renderError('导入失败：' . $e->getMessage());
        }
    }
    
    /**
     * 取消导入
     */
    public function cancel()
    {
        $filePath = session('import_file_path');
        
        // 删除临时文件
        if ($filePath && file_exists($filePath)) {
            unlink($filePath);
        }
        
        // 清除session
        session('import_file_path', null);
        session('import_order_numbers', null);
        
        return $this->renderSuccess('已取消');
    }
}

            $.post('/store/package.payment_import/confirm', {}, function(res) {
                if (res.code == 1) {
                    layer.msg('导入成功，共更新 ' + res.data.affected_rows + ' 个订单', {icon: 1});
                    setTimeout(function() {
                        location.reload();
                    }, 1500);
                } else {
                    layer.msg(res.msg, {icon: 2});
                }
            });
            layer.close(index);
        });
    });
    
    // 取消导入
    $('#cancel-btn').on('click', function() {
        $.post('/store/package.payment_import/cancel', {}, function(res) {
            location.reload();
        });
    });
});
```

## 六、使用流程

### 6.1 操作步骤

```
1. 财务打开原始Excel表格
   ↓
2. 将已支付的订单行着色（黄色、绿色等任何颜色）
   ↓
3. 保存Excel文件
   ↓
4. 在系统中进入"历史订单支付状态导入"页面
   ↓
5. 上传Excel文件
   ↓
6. 系统解析并显示预览：
   - 可更新订单：X个
   - 已支付订单：X个（跳过）
   - 未找到订单：X个（提示）
   ↓
7. 财务确认预览结果
   ↓
8. 点击"确认导入"
   ↓
9. 系统批量更新订单状态
   ↓
10. 显示导入结果
```

### 6.2 注意事项

1. **备份数据**：导入前建议备份数据库
2. **测试导入**：先用小批量数据测试
3. **检查结果**：导入后检查"未找到订单"列表
4. **不可逆操作**：导入后无法自动回滚，需手动修正
