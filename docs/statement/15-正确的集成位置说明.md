## 后端数据关联

需要在 `source/application/store/controller/tr_order/Index.php` 的数据查询中添加账单关联。

### 方法1: 在控制器中添加

```php
// 在获取订单列表后
$list = $orderModel->getList($params);

// 手动关联账单数据
foreach ($list as &$order) {
    if ($order['statement_id']) {
        $order['statement'] = \app\store\model\Statement::find($order['statement_id']);
    }
}
```

### 方法2: 在模型中添加关联

在订单模型中添加 `statement` 关联方法：

```php
public function statement()
{
    return $this->belongsTo('app\store\model\Statement', 'statement_id');
}
```

然后在查询时使用 `with(['statement'])`。

## 注意事项

1. **tr_order vs package**: 
   - `tr_order` 是集运订单（打包后的订单）
   - `package` 是包裹（未打包的包裹）
   - 账单功能应该用于集运订单，不是包裹

2. **数据模型**: 
   - 需要确认tr_order使用的是哪个模型
   - 可能是 `TrOrder` 模型或其他名称

3. **字段名称**: 
   - 确认订单表中的字段名称（可能不是 `member_id`）
   - 可能需要调整组件中的字段引用
