# 多格式账单生成方案

## 需求
支持生成多种格式的账单：
1. Excel格式（已实现）
2. PDF格式（待实现）
3. JPG图片格式（待实现）

## 技术方案

### 1. PDF生成
**推荐方案：mPDF**
- 安装：`composer require mpdf/mpdf`
- 优点：支持中文、支持CSS样式、易于使用
- 实现：HTML → PDF

### 2. JPG生成
**推荐方案：wkhtmltoimage**
- 安装：需要系统级安装 wkhtmltopdf/wkhtmltoimage
- Windows: 下载安装包 https://wkhtmltopdf.org/downloads.html
- 优点：完美渲染HTML/CSS、支持Tailwind
- 实现：HTML → JPG

**备选方案：Puppeteer（需要Node.js）**
- 需要安装Node.js和Puppeteer
- 通过PHP调用Node.js脚本

### 3. 数据库变更
```sql
ALTER TABLE yoshop_statement 
ADD COLUMN pdf_path VARCHAR(255) DEFAULT '' COMMENT 'PDF文件路径',
ADD COLUMN jpg_path VARCHAR(255) DEFAULT '' COMMENT 'JPG文件路径';
```

### 4. 文件结构
```
source/application/store/service/
├── statement/
│   ├── HtmlTemplateService.php  (已创建)
│   ├── PdfService.php           (待创建)
│   └── ImageService.php         (待创建)
```

### 5. 实现步骤
1. 安装mPDF库
2. 创建PdfService服务
3. 创建ImageService服务
4. 修改StatementService，在生成账单时同时生成PDF和JPG
5. 更新数据库表结构
6. 修改前端，添加PDF和JPG下载按钮

## 快速实现（最小化方案）

如果不想安装额外依赖，可以：
1. **PDF**: 使用PHP内置的TCPDF（ThinkPHP可能已包含）
2. **JPG**: 先不实现，或者只提供HTML预览页面
