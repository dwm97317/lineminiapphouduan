### 4. JavaScript功能

**位置**: 文件末尾，删除元素代码之后

**功能**: 生成账单

**代码逻辑**:
1. 获取选中的订单ID
2. 验证是否选择了订单
3. 从复选框的`data-member-id`属性获取客户ID
4. 验证所有选中订单是否属于同一客户
5. 弹出确认对话框
6. 调用API生成账单
7. 成功后刷新页面

**关键代码**:
```javascript
$('#j-create-statement').on('click', function () {
    var selectIds = checker.getCheckSelect();
    
    if (selectIds.length === 0) {
        layer.msg('请先选择订单');
        return;
    }
    
    // 验证是否同一客户
    var memberIds = [];
    $('input[name="checkIds"]:checked').each(function() {
        var memberId = $(this).data('member-id');
        if (memberId && memberIds.indexOf(memberId) === -1) {
            memberIds.push(memberId);
        }
    });
    
    if (memberIds.length > 1) {
        layer.msg('只能选择同一客户的订单生成账单');
        return;
    }
    
    // 确认并生成
    layer.confirm('确定为选中的 ' + selectIds.length + ' 个订单生成账单吗？', {
        title: '生成账单',
        btn: ['确定', '取消']
    }, function(index) {
        $.ajax({
            type: 'POST',
            url: "<?= url('package.statement/create') ?>",
            data: {
                package_ids: selectIds,
                member_id: memberIds[0]
            },
            dataType: "json",
            success: function(res) {
                if (res.code == 1) {
                    layer.msg('账单生成成功', {icon: 1});
                    setTimeout(function() {
                        location.reload();
                    }, 1500);
                } else {
                    layer.msg(res.msg, {icon: 2});
                }
            }
        });
        layer.close(index);
    });
});
```

## 用户操作流程

### 生成账单流程

1. 用户在包裹列表页面筛选订单（可选）
2. 勾选需要生成账单的订单（必须是同一客户）
3. 点击"生成账单"按钮
4. 系统验证：
   - 是否选择了订单
   - 订单是否属于同一客户
5. 弹出确认对话框
6. 用户确认后，系统调用API生成账单
7. 生成成功后，页面自动刷新
8. 订单的"账单信息"列显示账单编号和支付状态

### 查看账单流程

1. 在订单列表中，点击"账单信息"列的账单编号
2. 跳转到账单详情页面
3. 或者点击"账单列表"按钮，查看所有账单

### 3. 筛选逻辑
需要在控制器中处理`statement_status`参数：
```php
$statementStatus = $request->get('statement_status');
if ($statementStatus == 'unbilled') {
    $query->where('statement_id', 'null');
} elseif ($statementStatus == 'unpaid') {
    $query->whereNotNull('statement_id')
          ->whereHas('statement', function($q) {
              $q->where('pay_status', 1);
          });
} elseif ($statementStatus == 'paid') {
    $query->whereNotNull('statement_id')
          ->whereHas('statement', function($q) {
              $q->where('pay_status', 2);
          });
}
```

1. **客户验证**: 必须确保选中的订单属于同一客户
