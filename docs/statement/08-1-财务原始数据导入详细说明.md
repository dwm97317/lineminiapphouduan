# 财务原始数据导入 - 详细说明

## 一、实际数据格式

### 1.1 Excel文件结构

**文件特点**：
- 包含多个Sheet，每个Sheet对应一个或多个客户
- 数据按时间顺序排列
- 使用颜色标记订单状态

**列结构**（从图片分析）：
```
A列：用户ID + 国家（如：23048泰国、25831泰国）
B列：重量（KG）
C列：国际单号（可能为空）
D列：日期（可选，如：2月13日）
```

**示例数据**：
```
| A列        | B列  | C列              | D列     |
|-----------|------|------------------|---------|
| 23048泰国  | 1.8  | 73596298537719   |         |
| 25831泰国  | 5.2  | 73596298743207   |         | (蓝色背景)
| 17136泰国  | 4.2  | 73596298537721   |         | (蓝色背景)
| 23273泰国  | 3    | 73596298668051   |         | (粉红色背景)
| 30424泰国  | 1.2  | 73596298668049   |         | (绿色背景)
| 5809越南   | 2.6  | SF512221791891   | 2月13日 | (绿色背景)
| 5809泰国   | 29.6 |                  |         | (绿色背景，无国际单号)
```

### 1.2 颜色标记含义

**蓝色背景**：
- 状态：已支付 + 已发送账单
- 操作：标记订单为已支付（is_pay=1）
- 操作：创建历史账单记录（statement_id）

**粉红色背景**：
- 状态：已发送账单（未支付）
- 操作：创建历史账单记录（statement_id）
- 操作：保持未支付状态（is_pay=0）

**绿色背景**：
- 状态：已支付（未发送账单）
- 操作：标记订单为已支付（is_pay=1）
- 操作：不创建账单记录

**无背景色（白色）**：
- 状态：未处理
- 操作：不做任何处理

### 1.3 特殊情况处理

**情况1：用户ID前有图片**
```
| [图片] 23048泰国 | 1.8 | 73596298537719 |
```
处理：提取文本内容，忽略图片

**情况2：国际单号为空**
```
| 5809泰国 | 29.6 | (空) |
```
处理：通过用户ID + 重量 + 日期范围匹配订单

**情况3：用户ID包含国家信息**
```
23048泰国 → 提取用户ID：23048
5809越南 → 提取用户ID：5809
```

## 二、订单匹配逻辑

### 2.1 匹配优先级

```
1. 通过国际单号精确匹配（如果有）
   ↓ 如果匹配失败或国际单号为空
   
2. 通过用户ID + 重量 + 日期范围模糊匹配
   ↓ 如果匹配到多个
   
3. 选择最接近的订单（按日期、重量差异排序）
```

### 2.2 匹配规则

**规则1：国际单号匹配**
```sql
SELECT * FROM yoshop_inpack 
WHERE express_num = '73596298537719'  -- 国际单号
  AND is_delete = 0
LIMIT 1;
```

**规则2：用户ID + 重量匹配**
```sql
SELECT * FROM yoshop_inpack 
WHERE member_id = 23048
  AND ABS(weight - 1.8) < 0.5  -- 重量误差±0.5KG
  AND entering_warehouse_time BETWEEN '2025-02-01' AND '2025-02-28'  -- 日期范围
  AND is_delete = 0
ORDER BY ABS(weight - 1.8) ASC, entering_warehouse_time DESC
LIMIT 1;
```

**规则3：用户ID + 重量 + 未匹配过**
```sql
SELECT * FROM yoshop_inpack 
WHERE member_id = 23048
  AND ABS(weight - 1.8) < 0.5
  AND statement_id IS NULL  -- 未出账
  AND is_pay = 0  -- 未支付
  AND is_delete = 0
ORDER BY entering_warehouse_time DESC
LIMIT 1;
```

## 三、数据处理流程

### 3.1 完整流程

```
1. 读取Excel文件的所有Sheet
   ↓
2. 遍历每个Sheet的每一行
   ↓
3. 提取数据：
   - 用户ID（去除国家后缀）
   - 重量
   - 国际单号（可能为空）
   - 日期（可选）
   - 背景色
   ↓
4. 根据背景色分类：
   - 蓝色 → 已支付已出账列表
   - 粉红色 → 已出账未支付列表
   - 绿色 → 已支付未出账列表
   - 白色 → 跳过
   ↓
5. 匹配订单：
   - 优先通过国际单号匹配
   - 无国际单号则通过用户ID+重量+日期匹配
   ↓
6. 批量更新订单状态：
   - 蓝色：is_pay=1, statement_id=历史账单ID
   - 粉红色：statement_id=历史账单ID
   - 绿色：is_pay=1
   ↓
7. 生成导入报告
```

### 3.2 PHP实现示例

```php
/**
 * 解析财务原始数据
 */
public function parseFinanceRawData($filePath)
{
    $spreadsheet = IOFactory::load($filePath);
    $allSheets = $spreadsheet->getAllSheets();
    
    $results = [
        'blue' => [],      // 已支付已出账
        'pink' => [],      // 已出账未支付
        'green' => [],     // 已支付未出账
        'unmatched' => []  // 无法匹配
    ];
    
    foreach ($allSheets as $sheet) {
        $sheetName = $sheet->getTitle();
        $highestRow = $sheet->getHighestRow();
        
        for ($row = 2; $row <= $highestRow; $row++) {
            // 提取数据
            $memberIdRaw = trim($sheet->getCell('A' . $row)->getValue());
            $weight = floatval($sheet->getCell('B' . $row)->getValue());
            $internationalNo = trim($sheet->getCell('C' . $row)->getValue());
            $date = $sheet->getCell('D' . $row)->getValue();
            
            // 提取用户ID（去除国家后缀）
            $memberId = $this->extractMemberId($memberIdRaw);
            
            if (!$memberId || !$weight) {
                continue;  // 跳过无效数据
            }
            
            // 检测背景色
            $color = $this->detectCellColor($sheet, $row);
            
            if ($color == 'white') {
                continue;  // 跳过无背景色的行
            }
            
            // 匹配订单
            $package = $this->matchPackage($memberId, $weight, $internationalNo, $date);
            
            $item = [
                'sheet_name' => $sheetName,
                'row' => $row,
                'member_id' => $memberId,
                'weight' => $weight,
                'international_no' => $internationalNo,
                'date' => $date,
                'color' => $color,
                'package' => $package
            ];
            
            if ($package) {
                $results[$color][] = $item;
            } else {
                $results['unmatched'][] = $item;
            }
        }
    }
    
    return $results;
}

/**
 * 提取用户ID（去除国家后缀）
 */
private function extractMemberId($raw)
{
    // 去除图片、空格等
    $text = strip_tags($raw);
    $text = trim($text);
    
    // 提取数字部分（用户ID）
    if (preg_match('/^(\d+)/', $text, $matches)) {
        return intval($matches[1]);
    }
    
    return null;
}

/**
 * 检测单元格背景色
 */
private function detectCellColor($sheet, $row)
{
    // 检查A列或B列的背景色
    foreach (['A', 'B'] as $col) {
        $cell = $sheet->getCell($col . $row);
        $style = $cell->getStyle();
        $fill = $style->getFill();
        $fillColor = $fill->getStartColor()->getRGB();
        
        if (!$fillColor || $fillColor == 'FFFFFF') {
            continue;
        }
        
        // 根据RGB值判断颜色
        // 蓝色：通常是 00B0F0 或类似
        // 粉红色：通常是 FFC7CE 或类似
        // 绿色：通常是 C6EFCE 或类似
        
        if ($this->isBlueColor($fillColor)) {
            return 'blue';
        } elseif ($this->isPinkColor($fillColor)) {
            return 'pink';
        } elseif ($this->isGreenColor($fillColor)) {
            return 'green';
        }
    }
    
    return 'white';
}

/**
 * 判断是否为蓝色
 */
private function isBlueColor($rgb)
{
    // 蓝色系：B值较高，R和G值较低
    $r = hexdec(substr($rgb, 0, 2));
    $g = hexdec(substr($rgb, 2, 2));
    $b = hexdec(substr($rgb, 4, 2));
    
    return $b > 200 && $b > $r && $b > $g;
}

/**
 * 判断是否为粉红色
 */
private function isPinkColor($rgb)
{
    // 粉红色系：R值较高，G和B值中等
    $r = hexdec(substr($rgb, 0, 2));
    $g = hexdec(substr($rgb, 2, 2));
    $b = hexdec(substr($rgb, 4, 2));
    
    return $r > 200 && $g > 150 && $b > 150;
}

/**
 * 判断是否为绿色
 */
private function isGreenColor($rgb)
{
    // 绿色系：G值较高
    $r = hexdec(substr($rgb, 0, 2));
    $g = hexdec(substr($rgb, 2, 2));
    $b = hexdec(substr($rgb, 4, 2));
    
    return $g > 200 && $g > $r && $g > $b;
}

/**
 * 匹配订单
 */
private function matchPackage($memberId, $weight, $internationalNo, $date)
{
    // 方式1：通过国际单号精确匹配
    if (!empty($internationalNo)) {
        $package = Package::where('express_num', $internationalNo)
            ->where('is_delete', 0)
            ->find();
        
        if ($package) {
            return $package;
        }
    }
    
    // 方式2：通过用户ID + 重量模糊匹配
    $dateRange = $this->getDateRange($date);
    
    $package = Package::where('member_id', $memberId)
        ->where('is_delete', 0)
        ->where(function($query) use ($weight) {
            // 重量误差±0.5KG
            $query->where('weight', '>=', $weight - 0.5)
                  ->where('weight', '<=', $weight + 0.5);
        })
        ->where('entering_warehouse_time', '>=', $dateRange['start'])
        ->where('entering_warehouse_time', '<=', $dateRange['end'])
        ->order('entering_warehouse_time', 'desc')
        ->find();
    
    return $package;
}

/**
 * 获取日期范围
 */
private function getDateRange($date)
{
    if ($date) {
        // 解析日期（如：2月13日）
        if (preg_match('/(\d+)月(\d+)日/', $date, $matches)) {
            $month = intval($matches[1]);
            $day = intval($matches[2]);
            $year = date('Y');
            
            return [
                'start' => "$year-$month-$day 00:00:00",
                'end' => "$year-$month-$day 23:59:59"
            ];
        }
    }
    
    // 默认：当月
    return [
        'start' => date('Y-m-01 00:00:00'),
        'end' => date('Y-m-t 23:59:59')
    ];
}

/**
 * 批量更新订单状态
 */
public function batchUpdateFromRawData($results)
{
    $historyStatementId = $this->createHistoryStatement();
    
    Db::startTrans();
    try {
        // 处理蓝色（已支付已出账）
        foreach ($results['blue'] as $item) {
            if ($item['package']) {
                Package::where('id', $item['package']['id'])
                    ->update([
                        'is_pay' => 1,
                        'pay_time' => date('Y-m-d H:i:s'),
                        'statement_id' => $historyStatementId
                    ]);
            }
        }
        
        // 处理粉红色（已出账未支付）
        foreach ($results['pink'] as $item) {
            if ($item['package']) {
                Package::where('id', $item['package']['id'])
                    ->update([
                        'statement_id' => $historyStatementId
                    ]);
            }
        }
        
        // 处理绿色（已支付未出账）
        foreach ($results['green'] as $item) {
            if ($item['package']) {
                Package::where('id', $item['package']['id'])
                    ->update([
                        'is_pay' => 1,
                        'pay_time' => date('Y-m-d H:i:s')
                    ]);
            }
        }
        
        Db::commit();
        
        return [
            'blue_count' => count($results['blue']),
            'pink_count' => count($results['pink']),
            'green_count' => count($results['green']),
            'unmatched_count' => count($results['unmatched'])
        ];
        
    } catch (\Exception $e) {
        Db::rollback();
        throw $e;
    }
}

/**
 * 创建历史账单记录
 */
private function createHistoryStatement()
{
    $statement = Statement::create([
        'statement_no' => 'HISTORY_' . date('YmdHis'),
        'member_id' => 0,
        'member_name' => '历史数据导入',
        'start_date' => date('Y-m-01'),
        'end_date' => date('Y-m-t'),
        'status' => Statement::STATUS_CONFIRMED,
        'pay_status' => Statement::PAY_STATUS_PAID,
        'wxapp_id' => $this->wxapp_id
    ]);
    
    return $statement['id'];
}
```

## 五、注意事项

1. **颜色识别准确性**：不同Excel版本颜色RGB值可能略有差异
2. **重量匹配误差**：建议设置±0.5KG的误差范围
3. **日期范围**：如果没有明确日期，使用当月作为范围
4. **国际单号为空**：需要更严格的匹配条件（用户ID+重量+日期）
5. **历史账单**：为导入的历史数据创建统一的历史账单记录

## 七、总结

财务原始数据导入是一个复杂的功能，需要：
1. 识别多种颜色标记
2. 处理特殊格式（图片、空值）
3. 智能匹配订单（国际单号或用户ID+重量）
4. 根据颜色执行不同的更新逻辑

建议分阶段实施：
- 第一阶段：实现基本的颜色识别和国际单号匹配
- 第二阶段：实现用户ID+重量的模糊匹配
- 第三阶段：优化匹配算法，提高准确率
