# 订单转单功能修复说明

## 问题
点击"转单"按钮更换承运商后，承运商名称无法更新。

## 原因
代码中的字段白名单缺少 `tt_number` 和 `transfer` 字段，导致表单提交的承运商信息被过滤掉。

## 修复
在 `source/application/store/model/Inpack.php` 第377行，添加缺失的字段：

```php
// 修改前
$field = ['line_id','length',...,'t_order_sn'];

// 修改后
$field = ['line_id','length',...,'t_order_sn','tt_number','transfer'];
```

## 测试


应该看到：
```
✓ tt_number 字段存在
✓ transfer 字段存在
✓ 可以查询承运商
```

## 使用方法
1. 进入订单列表
2. 点击已发货订单的"转单"按钮
3. 选择新的承运商
4. 输入新的国际单号
5. 提交

## 验证
查询数据库确认更新：
```sql
SELECT order_sn, t2_name, t2_number, t2_order_sn 
FROM yoshop_inpack 
WHERE id = {订单ID};
```

`t2_name` 应该显示新的承运商名称。
