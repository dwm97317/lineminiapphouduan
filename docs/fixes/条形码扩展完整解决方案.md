## 🔧 方案 2: 安装 Imagick（不推荐，仅供参考）

### 为什么不推荐？
- 安装复杂，需要多个步骤
- 需要下载额外的 ImageMagick 软件
- 版本兼容性问题多
- GD 扩展已经足够使用

### 如果确实需要安装

#### 步骤 1: 下载 ImageMagick
1. 访问: https://imagemagick.org/script/download.php#windows
2. 下载对应版本：
   - PHP 7.3 + x64 → 下载 `ImageMagick-7.x.x-Q16-x64-dll.exe`
   - PHP 7.3 + x86 → 下载 `ImageMagick-7.x.x-Q16-x86-dll.exe`
3. 安装时勾选 "Install development headers and libraries"

#### 步骤 2: 下载 PHP Imagick 扩展
1. 访问: https://pecl.php.net/package/imagick
2. 或访问: https://windows.php.net/downloads/pecl/releases/imagick/
3. 下载匹配的版本：
   - PHP 7.3
   - Thread Safe (TS) 或 Non Thread Safe (NTS)
   - x64 或 x86

#### 步骤 3: 安装扩展
1. 解压下载的 zip 文件
2. 复制 `php_imagick.dll` 到 `D:\php7\ext\`
3. 复制其他 DLL 文件到 `D:\php7\`

#### 步骤 4: 配置 php.ini
```ini
extension=imagick
```

#### 步骤 5: 重启 PHP 服务

### 常见问题
- **找不到 DLL**: 确保 ImageMagick 的 bin 目录在系统 PATH 中
- **版本不匹配**: PHP 版本、架构、TS/NTS 必须完全匹配
- **依赖缺失**: 可能需要安装 Visual C++ Redistributable

## 📝 技术细节

### GD vs Imagick vs SVG

| 特性 | GD | Imagick | SVG |
|------|----|---------|----|
| 安装难度 | 简单 | 复杂 | 无需安装 |
| 性能 | 快 | 中等 | 最快 |
| 文件大小 | 中等 | 大 | 最小 |
| 质量 | 好 | 最好 | 矢量（完美） |
| 兼容性 | 高 | 中 | 高 |
| 打印支持 | ✅ | ✅ | ⚠️ 部分支持 |

### 代码逻辑

```php
// BarCodeService.php
public function Generator($code) {
    // 1. 检查是否支持 PNG
    $canUsePNG = extension_loaded('gd') || extension_loaded('imagick');
    
    if ($canUsePNG) {
        // 使用 PNG 格式
        return $this->generatePNG($code);
    } else {
        // 降级到 SVG 格式
        return $this->generateSVG($code);
    }
}
```
